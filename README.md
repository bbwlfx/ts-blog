## 技术栈
- @rematch（react状态管理，更好更简洁的redux）
- Typescript（静态检查）
- React（View）
- react-loadable（按需加载）
- @react/router（路由）
- Webpack（打包工具）
- Koa2（路由和渲染）
- SSR（加速渲染和SEO优化）
- Eslint（语法检测）
- husky（precommit hooks，对提交代码进行处理）
- antd（UI库）
- sequelize（连接数据库Mysql）
## Start for Mac
```
$ npm install
$ npm run startfe
$ npm run start
```
